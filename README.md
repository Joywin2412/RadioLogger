## Features

- Amateur Radio "Ham" Logging
- Shortwave Radio Logging
- MW DX Logging
- Scanner Logging
- Contest Logging
- QSL Card Database

## Requirements

- Node.js version (e.g., v14.0.0 or higher)
  We suggest using the latest LTS version.

- MongoDB version (e.g., v4.0.0 or higher)
